# 區塊勢 Blocktrend - GCC Demo Day Presentation

華語區塊鏈知識的公共基礎建設

## 專案說明

這是區塊勢 (Blocktrend) 為 GCC Public Goods Grant Cohort 2 Demo Day 製作的簡報網站。

## 功能特色

- 📱 響應式設計，支援桌面與行動裝置
- ⌨️ 鍵盤導航（方向鍵、空白鍵）
- 👆 觸控滑動手勢支援
- 🎨 現代化 UI 設計
- 📊 進度條顯示

## 使用方式

1. 直接在瀏覽器開啟 `index.html`
2. 使用方向鍵或點擊左右按鈕切換投影片
3. 在行動裝置上可滑動切換

## GitHub Pages 線上預覽

若已啟用 GitHub Pages，可在此查看：**https://astrohsu.github.io/gcc_cohort2/**

## 投影片內容

1. 封面
2. 真實存在的公共問題
3. 不可替代性與解決方案
4. 影響力與驗證數據
5. 未來願景：從金融到經濟
6. 資金預算與里程碑
